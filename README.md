# Nitshift

A multi-monitor brightness management application for Windows.

## Features

- **Native & Software Brightness** - Hardware DDC-CI control or software overlay mode
- **Multi-Monitor Support** - Individual brightness control per display
- **Brightness Presets** - Save and switch between custom configurations
- **Scheduling** - Automatic adjustments based on time or sunrise/sunset
- **System Tray** - Unobtrusive background operation

## Installation

```bash
npm install
```

## Usage

```bash
# Development
npm run dev

# Build
npm run build

# Package installer
npm run make
```

## Requirements

- Windows
- Node.js
- Monitors with DDC-CI support (for native brightness control)

## Tech Stack

- Electron
- React
- TypeScript
- Redux Toolkit
- Material-UI
